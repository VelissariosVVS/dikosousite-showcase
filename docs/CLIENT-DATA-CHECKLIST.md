# Checklist δεδομένων νέου πελάτη

Για να δημιουργηθεί ολοκληρωμένο client website, συγκέντρωσε τα παρακάτω.

## Βασικά στοιχεία

- Επωνυμία και διακριτικός τίτλος
- Επάγγελμα / ειδικότητα
- Τηλέφωνο, email και WhatsApp
- Πλήρης διεύθυνση και περιοχές εξυπηρέτησης
- Ωράριο
- Google Maps link
- Social media links
- Domain που υπάρχει ή επιθυμητό domain

## Εμπορικό περιεχόμενο

- Κύριες υπηρεσίες ή προϊόντα
- Τι διαφοροποιεί την επιχείρηση
- Ιδανικός πελάτης
- Βασικό call-to-action: κλήση, ραντεβού, κράτηση, παραγγελία ή επίσκεψη
- Συχνές ερωτήσεις
- Τρόποι πληρωμής, parking, πρόσβαση, delivery ή booking πληροφορίες

## Branding

- Logo σε SVG ή PNG υψηλής ανάλυσης
- Χρώματα brand
- Φωτογραφίες χώρου, ομάδας και εργασιών
- Υλικό πριν/μετά μόνο όταν επιτρέπεται από τον κλάδο και υπάρχει συναίνεση
- Προτιμώμενο template

## Reviews

- Επιλεγμένες πραγματικές αξιολογήσεις
- Όνομα/αρχικό όπως επιτρέπεται να εμφανίζεται
- Πηγή και ημερομηνία
- Άδεια χρήσης όπου απαιτείται

Μην κατασκευάζεις ψεύτικες αξιολογήσεις. Μην αντιγράφεις μαζικά περιεχόμενο τρίτων χωρίς έλεγχο δικαιωμάτων και ακρίβειας.

## Local SEO

- Ακριβής κατηγορία Google Business Profile
- Κύριες περιοχές στόχευσης
- Κύριες υπηρεσίες ανά περιοχή
- Συνήθεις φράσεις που χρησιμοποιούν οι πελάτες
- Ανταγωνιστές που εμφανίζονται στα τοπικά αποτελέσματα
- Υπάρχον website και Search Console, εφόσον υπάρχουν

## Παράδοση σε εμένα για ανάπτυξη

Μπορείς να στείλεις τα δεδομένα σε μορφή:

```text
Επωνυμία:
Κλάδος:
Template:
Περιοχή:
Τηλέφωνο:
Email:
Ωράριο:
Υπηρεσίες:
Βασικά πλεονεκτήματα:
Google Maps:
Social links:
Reviews:
Φωτογραφίες:
Domain:
Κύριος στόχος site:
```

Με αυτά μπορούν να παραχθούν προσαρμοσμένο περιεχόμενο, pages, metadata, LocalBusiness schema και οδηγίες deployment.
